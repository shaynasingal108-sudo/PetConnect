# PetConnect

This project was created using Lovable, a no-code platform recommended
by GradCircle for this competition.

## Live Demo
https://furry-friends-square.lovable.app/auth

## Overview
PetConnect is a website used to allow pet owners to connect and support one another efficiently. It allows users to communicate, grow, and educate themselves about their pets through an organized, trustowrthy, helpful manner.

## Implementation
The application was built using Lovable, which handles the underlying
source code and deployment. This repository documents the project and
provides access to the live version.
